\subsection{Detailed Description}
Diff\-Problem\-First\-Second class implementation. Describes an initial value for a partitioned dynamic system with an O\-D\-E or D\-A\-E description. The system is composed by two subsystem of orders 1 and 2, respectively.

This is the base file of lmx\-\_\-diff systems' manipulation and solution.
