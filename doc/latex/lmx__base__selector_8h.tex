\subsection{Detailed Description}
This file contains set-\/get functions for switching between the \hyperlink{classMatrix}{Matrix}, \hyperlink{classVector}{Vector} and Linear Solvers types. Selector class is not usually instantiated. Instead, its friend functions are used by \hyperlink{classMatrix}{Matrix}, \hyperlink{classVector}{Vector} and \hyperlink{classLinearSystem}{Linear\-System} objects for getting the type that is being used.
