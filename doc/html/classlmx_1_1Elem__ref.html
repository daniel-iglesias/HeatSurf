Detailed Description

template<typename T>
class lmx::Elem_ref< T >

Template class Elem_ref.

This class is used to gain access for read and write purposes in any data structure derived from the Data class.

Parameters
typePointer to the matrix or vector structure that is a Data derived class.
valueCopy of the numerical value read.
iRow where the original value is stored.
jColumn where the original value is stored.
