template<typename Sys, typename T = double>
class lmx::DiffProblem< Sys, T >

Template class DiffProblem. Implementation for ODE system solvers.

This class implements methods for defining and solving initial value problems described by a TotalDiff class' derivided object, and initial conditions in the form , .
