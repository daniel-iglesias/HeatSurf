template<typename Sys, typename T = double>
class lmx::DiffProblemDouble< Sys, T >

Template class DiffProblemDouble. Implementation for ODE system solvers.

This class implements methods for defining and solving partitioned initial value problems described by a TotalDiff class' derivided object, and initial conditions in the form , .
