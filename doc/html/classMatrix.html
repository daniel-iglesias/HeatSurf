Detailed Description

template<typename T>
class Matrix< T >

Template class Matrix.

This class permits the creation of matrix objects. A Matrix object owns two parameters, nrows and mcolumns, that store the dimension of the matrix container. The data is stored in an atribute (*type_matrix) that points to some class which derives from the Data_mat class.

Parameters
mrowsThe number of rows of the Data Container (Data_mat).
ncolumnsThe number of columns of the Data Container (Data_mat).
referenceAn Elem_ref object for r/w data access.
*type_matrixThe pointer to the matrix data container.
Author
Daniel Iglesias.
