Detailed Description

DiffProblemFirstSecond class implementation.

Describes an initial value for a partitioned dynamic system with an ODE or DAE description. The system is composed by two subsystem of orders 1 and 2, respectively.

This is the base file of lmx_diff systems' manipulation and solution.
