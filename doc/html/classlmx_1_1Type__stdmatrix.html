Detailed Description

template<typename T>
class lmx::Type_stdmatrix< T >

Template class Type_stdmatrix.

This class implements the methods defined in virtual class data_mat for dense matrix built with std vectors.

Parameters
contentsCorresponds to a type std::vector< std::vector<T> >.
